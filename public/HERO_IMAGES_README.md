# Hero Carousel Images

Please add the following images to the `/public` folder for the hero carousel to work properly:

## Required Images

1. **hero-slide-1.jpg** (1920x1080)
   - Theme: Cosmic destiny, stars, vedic astrology
   - Suggested content: Cosmic background with stars, planets, or astrological symbols

2. **hero-slide-2.jpg** (1920x1080)
   - Theme: Kundli matching, couples, relationships
   - Suggested content: Beautiful couple imagery or romantic cosmic background

3. **hero-slide-3.jpg** (1920x1080)
   - Theme: Daily horoscope, zodiac signs
   - Suggested content: Zodiac wheel, celestial bodies, or horoscope imagery

## Image Specifications

- **Dimensions**: 1920x1080 pixels (16:9 aspect ratio)
- **Format**: JPG or PNG
- **Location**: Place directly in `/public` folder
- **Naming**: Must match exact filenames listed above

## Image Sources

You can source images from:
- Stock photo websites (Unsplash, Pexels, Freepik)
- Custom designed images
- AI-generated images (Midjourney, DALL-E)

## Important Notes

- Ensure images are optimized for web use
- Keep file sizes under 500KB for better performance
- Images should have good contrast on the left side where text overlays
- Consider using images with darker left side or subjects positioned on the right
